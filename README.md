Starts a session in the Matlab Command window. The user can type
questions and receives responses from the language model.

Feartures:
- Model: GPT-5.5
- Word limit: 2000
- Error handling: Api errors are caught and displayed without crashing

Requirements:
- Matlab R2025b or later
- Matlab LLM Toolbox
- open API key set as environment varible:
  On Windows: OPEN_API_KEY = Your API Key
